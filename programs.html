1. Hello World

🟢 Beginner

Problem: Write a C program to print "Hello, World!" on the screen.

Algorithm

  1. Start the program inside main().
  2. Use printf() to display the message.
  3. Return 0 to indicate successful execution.

C Program

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Sample Output

Hello, World!

Explanation

#include <stdio.h> imports the standard input/output library so we can use printf(). The main() function is the entry point of every C program. \n moves the cursor to a new line.

Concepts Used

stdio.h printf main()
🧠
Try Yourself: Modify the program to print your name on the next line.

2. Print Name

🟢 Beginner

Problem: Write a C program that takes the user's name as input and prints a greeting.

Algorithm

  1. Declare a char array to hold the name.
  2. Use scanf() to read the name from the user.
  3. Print a greeting using printf().

C Program

#include <stdio.h>

int main() {
    char name[50];
    printf("Enter your name: ");
    scanf("%s", name);
    printf("Hello, %s! Welcome to C Programming.\n", name);
    return 0;
}

Sample Output

Enter your name: Shyam
Hello, Shyam! Welcome to C Programming.

Explanation

char name[50] reserves space for 50 characters. scanf("%s", name) reads a word (no spaces). The %s format specifier inside printf is replaced with the entered name.

Concepts Used

char array scanf %s
🧠
Try Yourself: Modify it to also ask for age and print "Shyam is 20 years old".

3. Addition of Two Numbers

🟢 Beginner

Problem: Write a C program to add two integers entered by the user and display the sum.

Algorithm

  1. Declare three integer variables: a, b, sum.
  2. Read a and b using scanf().
  3. Compute sum = a + b.
  4. Print the result.

C Program

#include <stdio.h>

int main() {
    int a, b, sum;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    sum = a + b;
    printf("Sum = %d\n", sum);
    return 0;
}

Sample Output

Enter two numbers: 12 18
Sum = 30

Explanation

The & symbol gives the address of a variable so scanf can store the input there. %d is the format specifier for a signed decimal integer.

Concepts Used

int scanf %d arithmetic
🧠
Try Yourself: Extend the program to add three numbers instead of two.

4. Swap Two Numbers

🟢 Beginner

Problem: Write a C program to swap two integers using a temporary variable.

Algorithm

  1. Read two numbers into a and b.
  2. Copy the value of a into temp.
  3. Assign b to a.
  4. Assign temp to b.
  5. Print the swapped values.

C Program

#include <stdio.h>

int main() {
    int a, b, temp;
    printf("Enter a and b: ");
    scanf("%d %d", &a, &b);

    printf("Before swap: a = %d, b = %d\n", a, b);

    temp = a;
    a = b;
    b = temp;

    printf("After swap:  a = %d, b = %d\n", a, b);
    return 0;
}

Sample Output

Enter a and b: 5 9
Before swap: a = 5, b = 9
After swap:  a = 9, b = 5

Explanation

Without temp we would lose one value. The temporary variable holds the original a while we overwrite it with b.

Concepts Used

variables assignment temp variable
🧠
Try Yourself: Swap the same two numbers without using a third variable (use + and -).

5. Area of Circle

🟢 Beginner

Problem: Write a C program to calculate the area of a circle given its radius. (Use π = 3.14159)

Algorithm

  1. Read the radius into a float.
  2. Apply formula area = π × r × r.
  3. Display the area.

C Program

#include <stdio.h>

int main() {
    float radius, area;
    const float PI = 3.14159;

    printf("Enter radius: ");
    scanf("%f", &radius);

    area = PI * radius * radius;

    printf("Area of circle = %.2f\n", area);
    return 0;
}

Sample Output

Enter radius: 7
Area of circle = 153.94

Explanation

const makes PI read-only. %.2f prints the float rounded to 2 decimal places. Use float (or double) for decimal numbers.

Concepts Used

float const %f geometry
🧠
Try Yourself: Also calculate the circumference of the circle in the same program.

6. Simple Interest

🟢 Beginner

Problem: Calculate Simple Interest given Principal (P), Rate (R) and Time (T) in years.
Formula: SI = (P × R × T) / 100

Algorithm

  1. Read P, R, T as float.
  2. Apply formula SI = (P × R × T) / 100.
  3. Display result.

C Program

#include <stdio.h>

int main() {
    float P, R, T, SI;
    printf("Enter Principal, Rate, Time: ");
    scanf("%f %f %f", &P, &R, &T);

    SI = (P * R * T) / 100;

    printf("Simple Interest = %.2f\n", SI);
    return 0;
}

Sample Output

Enter Principal, Rate, Time: 1000 5 2
Simple Interest = 100.00

Explanation

The * and / are arithmetic operators. Operator precedence makes the parentheses crucial — without them, P × R would be divided by 100 × T instead.

Concepts Used

float formula arithmetic
🧠
Try Yourself: Also compute the total amount (Principal + Interest) and display it.

7. Even or Odd

🟢 Beginner

Problem: Write a C program to check whether a given integer is even or odd.

Algorithm

  1. Read an integer n.
  2. If n % 2 == 0, it's even; otherwise it's odd.
  3. Print the result.

C Program

#include <stdio.h>

int main() {
    int n;
    printf("Enter an integer: ");
    scanf("%d", &n);

    if (n % 2 == 0)
        printf("%d is Even\n", n);
    else
        printf("%d is Odd\n", n);

    return 0;
}

Sample Output

Enter an integer: 17
17 is Odd

Explanation

The modulo operator % returns the remainder. Any integer evenly divisible by 2 leaves remainder 0, so it's even.

Concepts Used

if-else modulus % conditional
🧠
Try Yourself: Modify it to print whether the number is positive, negative, or zero as well.

8. Largest of Two Numbers

🟢 Beginner

Problem: Write a C program to find the largest of two numbers entered by the user.

Algorithm

  1. Read two integers a and b.
  2. If a > b, a is largest.
  3. Else, b is largest.

C Program

#include <stdio.h>

int main() {
    int a, b;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);

    if (a > b)
        printf("%d is the largest.\n", a);
    else
        printf("%d is the largest.\n", b);

    return 0;
}

Sample Output

Enter two numbers: 42 27
42 is the largest.

Explanation

The relational operator > returns 1 (true) or 0 (false), which the if statement uses to choose the branch.

Concepts Used

if-else relational comparison
🧠

9. Factorial

🟢 Beginner

Problem: Calculate the factorial of a number n. (n! = 1 × 2 × 3 × … × n)

Algorithm

  1. Read n.
  2. Initialize fact = 1.
  3. Loop i from 1 to n, multiply fact *= i.
  4. Print fact.

C Program

#include <stdio.h>

int main() {
    int n, i;
    long long fact = 1;

    printf("Enter a number: ");
    scanf("%d", &n);

    for (i = 1; i <= n; i++) {
        fact = fact * i;
    }

    printf("Factorial of %d = %lld\n", n, fact);
    return 0;
}

Sample Output

Enter a number: 6
Factorial of 6 = 720

Explanation

Factorials grow fast, so we use long long to avoid overflow. The for loop runs exactly n times, multiplying each value into fact.

Concepts Used

for loop long long accumulator
🧠
Try Yourself: Modify the program to calculate factorial using recursion.

Problem: Print the multiplication table of a number up to 10.

Algorithm

}

Explanation

The loop runs 10 times. Each iteration prints one row of the table using formatted output.

for loop printf pattern

1. Largest Element in Array

🟡 Intermediate

Problem: Find the largest element in an array of integers.

Algorithm

  1. Declare array and read its size n.
  2. Read n elements.
  3. Assume max = arr[0].
  4. Loop through array, update max whenever a bigger value is found.
  5. Print max.

C Program

#include <stdio.h>

int main() {
    int arr[100], n, i, max;

    printf("Enter number of elements: ");
    scanf("%d", &n);

    printf("Enter %d elements:\n", n);
    for (i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    max = arr[0];
    for (i = 1; i < n; i++) {
        if (arr[i] > max)
            max = arr[i];
    }

    printf("Largest element = %d\n", max);
    return 0;
}

Sample Output

Enter number of elements: 5
Enter 5 elements: 12 45 7 33 28
Largest element = 45

Explanation

Arrays are indexed from 0. We initialise max with the first element, then compare every remaining element. If any element is bigger, we replace max.

Concepts Used

1D array for loop search
🧠
Try Yourself: Find the smallest element instead of the largest.

2. Sum of Array Elements

🟡 Intermediate

Problem: Calculate the sum and average of all elements in an array.

Algorithm

  1. Read array size n and elements.
  2. Loop and accumulate values in sum.
  3. Compute avg = sum / n (cast to float).
  4. Display both.

C Program

#include <stdio.h>

int main() {
    int arr[100], n, i, sum = 0;
    float avg;

    printf("Enter number of elements: ");
    scanf("%d", &n);

    printf("Enter %d elements:\n", n);
    for (i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
        sum += arr[i];
    }

    avg = (float)sum / n;

    printf("Sum = %d\n", sum);
    printf("Average = %.2f\n", avg);
    return 0;
}

Sample Output

Enter number of elements: 4
Enter 4 elements: 10 20 30 40
Sum = 100
Average = 25.00

Explanation

(float)sum converts sum from int to float so the division produces a decimal result instead of integer truncation.

Concepts Used

array type casting accumulator
🧠
Try Yourself: Also print the largest and smallest elements of the array.

3. Reverse a String

🟡 Intermediate

Problem: Reverse a string entered by the user without using the library function.

Algorithm

  1. Read a string into str.
  2. Find its length manually.
  3. Swap characters str[i] and str[len-1-i].
  4. Print the reversed string.

C Program

#include <stdio.h>

int main() {
    char str[100], temp;
    int i, len = 0;

    printf("Enter a string: ");
    scanf("%s", str);

    while (str[len] != '\0') len++;

    for (i = 0; i < len / 2; i++) {
        temp = str[i];
        str[i] = str[len - 1 - i];
        str[len - 1 - i] = temp;
    }

    printf("Reversed: %s\n", str);
    return 0;
}

Sample Output

Enter a string: hello
Reversed: olleh

Explanation

Strings in C are stored as char arrays ending with the null character '\0'. We swap characters from both ends moving towards the centre.

Concepts Used

string null terminator swap
🧠
Try Yourself: Reverse the string using a second array instead of swapping in place.

4. Palindrome String

🟡 Intermediate

Problem: Check whether a given string is a palindrome (reads the same forwards and backwards).

Algorithm

  1. Read the string.
  2. Find its length.
  3. Compare str[i] with str[len-1-i] for all i < len/2.
  4. If all match → palindrome; else → not palindrome.

C Program

#include <stdio.h>

int main() {
    char str[100];
    int i, len = 0, flag = 1;

    printf("Enter a string: ");
    scanf("%s", str);

    while (str[len] != '\0') len++;

    for (i = 0; i < len / 2; i++) {
        if (str[i] != str[len - 1 - i]) {
            flag = 0;
            break;
        }
    }

    if (flag)
        printf("\"%s\" is a palindrome.\n", str);
    else
        printf("\"%s\" is NOT a palindrome.\n", str);

    return 0;
}

Sample Output

Enter a string: level
"level" is a palindrome.

Explanation

A flag variable tracks the result. The moment a mismatch is found, we break out of the loop early to save time.

Concepts Used

string palindrome flag
🧠
Try Yourself: Make the check case-insensitive (e.g. "Level" should also be a palindrome).

1. Pointer Example

🟠 Advanced

Problem: Demonstrate the use of pointers — store the address of an integer and print its value through the pointer.

Algorithm

  1. Declare int x = 10 and a pointer int *p.
  2. Assign p = &x (store address of x).
  3. Print the address and the value via *p.

C Program

#include <stdio.h>

int main() {
    int x = 10;
    int *p = &x;

    printf("Value of x      = %d\n", x);
    printf("Address of x    = %p\n", &x);
    printf("Value of p      = %p\n", p);
    printf("Value at *p     = %d\n", *p);

    return 0;
}

Sample Output

Value of x      = 10
Address of x    = 0x7ffd4c3a
Value of p      = 0x7ffd4c3a
Value at *p     = 10

Explanation

A pointer stores a memory address. &x gives the address of x. *p "dereferences" the pointer — it goes to that address and reads the value there.

Concepts Used

pointer address-of & dereference *
🧠
Try Yourself: Modify the value of x through the pointer using *p = 20 and observe the change.

2. Swap Using Pointers

🟠 Advanced

Problem: Swap two numbers using a function and pointers (call by reference).

Algorithm

  1. Define function swap(int *a, int *b).
  2. Inside, swap values using a temp variable and dereferencing.
  3. In main(), pass addresses &x, &y.

C Program

#include <stdio.h>

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 5, y = 9;
    printf("Before swap: x = %d, y = %d\n", x, y);

    swap(&x, &y);

    printf("After swap:  x = %d, y = %d\n", x, y);
    return 0;
}

Sample Output

Before swap: x = 5, y = 9
After swap:  x = 9, y = 5

Explanation

This is called pass-by-reference. The function receives the addresses, so changes happen on the original variables — no copy is made.

Concepts Used

pointer pass by reference void function
🧠
Try Yourself: Write a swap() that swaps without using a temp variable (using XOR).

3. Student Record Using Structure

🟠 Advanced

Problem: Create a structure Student with roll, name and marks, then read and display one student's record.

Algorithm

  1. Define struct Student with roll, name, marks.
  2. Declare a struct Student s.
  3. Read values into its fields.
  4. Display the record.

C Program

#include <stdio.h>

struct Student {
    int roll;
    char name[50];
    float marks;
};

int main() {
    struct Student s;

    printf("Enter roll: ");
    scanf("%d", &s.roll);

    printf("Enter name: ");
    scanf("%s", s.name);

    printf("Enter marks: ");
    scanf("%f", &s.marks);

    printf("\n--- Student Record ---\n");
    printf("Roll : %d\n", s.roll);
    printf("Name : %s\n", s.name);
    printf("Marks: %.2f\n", s.marks);

    return 0;
}

Sample Output

Enter roll: 101
Enter name: Shyam
Enter marks: 88.5

--- Student Record ---
Roll : 101
Name : Shyam
Marks: 88.50

Explanation

A struct groups related variables of different types. We access fields using the dot (.) operator.

Concepts Used

struct user-defined type dot operator
🧠
Try Yourself: Add a field grade (char) and assign it based on marks (A/B/C).

4. Array of Structures

🟠 Advanced

Problem: Store and display records of multiple students using an array of structures.

Algorithm

  1. Define struct Student.
  2. Declare an array struct Student s[n].
  3. Loop to read each student's details.
  4. Loop again to display all records.

C Program

#include <stdio.h>

struct Student {
    int roll;
    char name[50];
    float marks;
};

int main() {
    struct Student s[50];
    int n, i;

    printf("Enter number of students: ");
    scanf("%d", &n);

    for (i = 0; i < n; i++) {
        printf("\nStudent %d\n", i + 1);
        printf("  Roll : ");  scanf("%d", &s[i].roll);
        printf("  Name : ");  scanf("%s", s[i].name);
        printf("  Marks: ");  scanf("%f", &s[i].marks);
    }

    printf("\n--- All Records ---\n");
    for (i = 0; i < n; i++) {
        printf("%d. %s (Roll %d) - %.2f\n",
               s[i].roll, s[i].name, s[i].roll, s[i].marks);
    }

    return 0;
}

Sample Output

Enter number of students: 2

Student 1
  Roll : 1
  Name : Asha
  Marks: 92
Student 2
  Roll : 2
  Name : Ravi
  Marks: 85

--- All Records ---
1. Asha (Roll 1) - 92.00
2. Ravi (Roll 2) - 85.00

Explanation

Combining arrays with structures lets us store many records of the same type. s[i].name accesses the i-th student's name field.

Concepts Used

struct array loop
🧠
Try Yourself: Also print the student with the highest marks after the loop.

5. File Write and Read

🟠 Advanced

Problem: Write a string to a text file, then read it back and display it on screen.

Algorithm

  1. Open data.txt in write mode with fopen("data.txt", "w").
  2. Write a string using fprintf.
  3. Close the file with fclose.
  4. Reopen in read mode ("r") and read with fscanf.
  5. Print the content and close again.

C Program

#include <stdio.h>

int main() {
    FILE *fp;
    char text[100];

    /* ---- WRITE ---- */
    fp = fopen("data.txt", "w");
    if (fp == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    fprintf(fp, "Hello from C Programming Hub!\n");
    fclose(fp);
    printf("Written to data.txt\n");

    /* ---- READ ---- */
    fp = fopen("data.txt", "r");
    if (fp == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    printf("Reading from data.txt:\n");
    while (fscanf(fp, "%[^\n]", text) != EOF) {
        printf("%s\n", text);
    }
    fclose(fp);

    return 0;
}

Sample Output

Written to data.txt
Reading from data.txt:
Hello from C Programming Hub!

Explanation

FILE is a structure in stdio.h. "w" creates / overwrites the file, "r" reads. Always fclose() when done. The format specifier %[^\n] reads until a newline.

Concepts Used

FILE * fopen / fclose fprintf / fscanf
🧠
Try Yourself: Append ("a" mode) the current date and time to the same file using fprintf.
